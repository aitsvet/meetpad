Задача: Проанализируй переданную агенду встречи, которая содержит цель, список вопросов/задач и общую длительность встречи.

Тебе нужно:
1. Выделить список вопросов/задач с их содержанием.
2. Определить общее время встречи (duration_minutes).

Отформатируй вывод строго в JSON формате и ничего кроме, например:
{
    "duration_minutes": число,
    "questions": [
        "вопрос 1",
        "вопрос 2",
        ...
    ]
}
